
# Care AI :-An AI Based System for Post-Treatment Lifestyle Optimization

AI-based personalized healthcare monitoring system that simulates real-time patient health data.

## Features
- Real-time sensor data simulation (HR, BP, SpO2, Sugar)
- Flask API for data access
- Multiple dataset support for training/testing

## Tech Stack
- Python
- Flask
- Waitress

## How to Run
1. Install dependencies:
   pip install flask waitress

2. Run server:
   python sensor_server.py

3. Open in browser:
   http://127.0.0.1:5000/
